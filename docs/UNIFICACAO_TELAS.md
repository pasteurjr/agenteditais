## 1. PRINCIPIOS DE DESIGN

### 1.1 Decisoes Arquiteturais (Correcoes Aplicadas)

| Aspecto | CARA SISTEMA.pdf (Original) | Decisao Final (Corrigido) |
|---------|----------------------------|---------------------------|
| **Layout** | Dois sidebars (esquerda + direita) | **Sidebar unico a esquerda** |
| **Menu Superior** | Empresa, Portfolio, Parametrizacoes separados | **Integrado no sidebar como secao "Configuracoes"** |
| **Tela de Validacao** | Todos elementos visiveis simultaneamente | **Progressive disclosure com abas** |
| **Scores** | Multiplas escalas (%, estrelas, texto) | **Padronizado: 0-100% com cores** |
| **Chat IA** | Duplicado (flutuante + contextual) | **Chat flutuante unico (Dr. Licita)** |
| **Botoes de Acao** | Dispersos na tela | **Barra fixa no rodape ou lateral** |

### 1.2 Padroes Visuais

```
CORES DE SCORE:
- Verde:   >= 80%  (Alta aderencia / Baixo risco)
- Amarelo: 50-79%  (Media aderencia / Atencao)
- Vermelho: < 50%  (Baixa aderencia / Alto risco)

HIERARQUIA VISUAL:
1. Score principal (destaque maximo)
2. Alertas criticos (vermelho)
3. Acoes primarias (botoes destacados)
4. Informacoes secundarias (texto menor)
```

### 1.3 Progressive Disclosure

Aplicar em telas complexas (especialmente Validacao):
- **Nivel 1**: Score geral + resumo IA + acoes primarias
- **Nivel 2**: Detalhamento por aba (Tecnico | Documental | Comercial | Riscos)
- **Nivel 3**: Dados completos sob demanda (modal ou expansao)

## 3. SISTEMA DE SCORES

### 3.1 Definicao Unificada (Fonte: CARA SISTEMA.pdf, Padronizado)

| Score | Descricao | Fonte de Dados | Visualizacao |
|-------|-----------|----------------|--------------|
| **Score de Aderencia Geral** | Consolidacao de todos os scores | Calculado | Circulo grande 0-100% |
| **Aderencia Tecnica** | Requisitos tecnicos do edital vs portfolio | Portfolio + Edital | Barra 0-100% |
| **Aderencia Comercial** | Regiao, prazo entrega, custos logisticos | Parametrizacoes | Barra 0-100% |
| **Aderencia Documental** | Documentos exigidos vs disponiveis | Empresa + Edital | Barra 0-100% |
| **Risco Juridico** | Historico do orgao, aditivos, acoes | Memoria corporativa | Barra 0-100% |
| **Viabilidade Logistica** | Distancia, prazo, custo de servir | Parametrizacoes | Barra 0-100% |
| **Atratividade Comercial** | Historico de precos, margem, concorrencia | Historico + PNCP | Barra 0-100% |

### 3.2 Fatores de Cada Score (Detalhamento do CARA SISTEMA.pdf)

#### Aderencia Tecnica
- Especificacoes do produto vs requisitos do edital
- Gaps identificados (ex: "Requisito 4.2: Torque Maximo - desvio de 3%")
- Certificacoes exigidas vs disponiveis

#### Aderencia Comercial
- Regiao de atuacao parametrizada
- Tempo de entrega viavel
- Tipo de edital desejado (Comodato, Venda, Aluguel, etc.)

#### Aderencia Documental
- Contrato Social atualizado
- AFE (Autorizacao de Funcionamento)
- CBPAD, CBPP, Corpo de Bombeiros
- Certidoes (negativas, capacidade tecnica)
- Balancos patrimoniais

#### Risco Juridico
- Historico de aditivos do orgao
- Acoes judiciais contra fornecedores
- Indicios de direcionamento
- Perfil do pregoeiro (rigoroso, etc.)

#### Viabilidade Logistica
- Distancia ate o orgao
- Frequencia de entrega
- Custo de servir

#### Atratividade Comercial
- Historico de precos praticados
- Preco predatorio identificado
- Historico de pagamentos do orgao
- Margem estimada
- Concorrente dominante
